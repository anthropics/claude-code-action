üìù Manual Setup

If the buttons above don't work, you can manually create the app:

  1. Copy the manifest JSON below:
{ "name": "Claude Code Custom App", "description": "Custom GitHub App for Claude Code Action", "url": "https://github.com/anthropics/claude-code-action", "hook_attributes": { "url": "https://example.com/github/webhook", "active": false }, "redirect_url": "https://github.com/settings/apps/new", "public": false, "default_permissions": { "contents": "write", "issues": "write", "pull_requests": "write", "actions": "read", "metadata": "read" }, "default_events": [ "issue_comment", "issues", "pull_request", "pull_request_review", "pull_request_review_comment" ] }
  1. Go to GitHub App Settings
  2. Look for "Create from manifest" option
  3. Paste the JSON and follow the prompts
